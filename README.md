# Convenient Store Admin

A monorepo for the convenient store admin application with separate frontend and backend services.

## Project Structure

```
.
├── frontend/     # Frontend application
├── backend/      # Backend API
└── README.md     # This file
```

## Prerequisites

- Node.js 18+
- npm or yarn

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
